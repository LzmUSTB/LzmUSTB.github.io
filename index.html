网页设计学习笔记

发表于2022-10-13字数统计1.1k

页面尺寸的选择

网页宽度&有效可视区

  • 网页宽度：进行页面设计时画板的宽度
  • 有效可视区：网页内容所占的宽度

Statcounter Global Stats上可以看到当前PC端显示器的流行尺寸，由此得出结论：

网页宽度：1920

有效可视区：900~1200

响应式布局

若不希望可视区占比太小，可以采用响应式布局使得网页在不同的尺寸下有不同的布局。

百分比

%

相对于父元素，或无父元素则相对于视窗

属性 百分比基准
width 基于父级的width
height 基于父级的height
margin(四个方向) 基于父级的width
padding(四个方向) 基于父级的width
left,,right 基于非static定位的父级的width
top,bottom 基于非static定位的父级的height

vw&vh

  • 1vw=1%视窗宽度
  • 1vh=1%视窗高度
  • vmin：vw和vh中较小值
  • vmax：vw和vh中较大值

rem(font size of the root element)

rem为相对于根元素字体大小的单位

媒体查询

flex布局

Flex布局中存在main axis(横向)和cross axis(纵向)两个轴，默认将元素在main axis从左向右排列

justify-content

可以决定容器中元素在主轴方向的位置

  • flex-start：将元素对齐主轴首端
  • flex-end：将元素对齐主轴末端
  • center：将元素居中
  • space-between：使第一个元素位于主轴首端，最后一个元素位于主轴末端
  • space-around：将元素等距排列，且第一个元素和最后一个元素到边界的距离相当于元素间距离的一半

align-items

可以决定容器中元素在副轴方向的位置

  • flex-start：将元素对齐副轴首端
  • flex-end：将元素对齐副轴末端
  • center：将元素居中
  • stretch：
  • baseline：

align-self

作用于flex布局内的元素。选项与align-items相同，用于具体修改某个元素的纵向布局。

flex-direction

可以修改主轴的方向

  • row：主轴为横向，从左向右
  • row-reverse：主轴为横向，从右向左
  • column：主轴为纵向，从上向下
  • column-reverse：主轴为纵向，从下向上

order

作用于flex布局内的元素。默认值为0，可选择任意整数值。数值大的元素将排列在后面。

Grid布局

fr：grid专有单位，表示比例（fraction）

grid-template

容器属性，用于定义栅格行列的宽度，数量

  • grid-template-columns
  • -rows
  • 【选项】repeat(重复次数,值)：
  • 【选项】minmax(最小值，最大值)
  • 【选项】auto
  • 【选项】网格线[...]
  • -areas：设定网格区域：

gap

容器属性，设置行，列的间距。

  • row-gap
  • column-gap
  • gap

grid-auto

grid-auto-flow

容器属性，设置自动排列的方向

  • 【选项1】row：按行排放（默认）
  • 【选项1】column：按列排放
  • 【选项2】dense：用后排项目填补空余部分

grid-auto-rows/columns

容器属性，设置多出项目的宽(column)和高(row)

对齐选项

-items

容器属性 ：决定项目在单元格内的对齐方式

  • justify-items：水平方向对齐
  • align-items：垂直方向对齐
    • start|end|center|stretch：stretch为撑满单元格
  • place-items：<align-items> <justify-items>

-content

容器属性：决定网格区域在父元素内的对齐方式

  • justify-content：水平方向对齐
  • align-content：垂直方向对齐
    • start|end|center|stretch
    • space-around
    • space-between
    • space-evenly：等间距分布（包括与边界的距离

项目属性

指定位置

  • grid-column/row-start/end

    ※简写:grid-column:1/3;

    ※跨行写法：grid-column-start:span 3;(占三列)

  • grid-area：指定所在区域

对齐

  • justify-self/align-self：设置项目在单元格内对齐方式

阴影

box-shadow: offset-x(x偏移) offset-y(y偏移) blur(模糊半径) spread(扩展半径) color(颜色) position(inset内阴影);
