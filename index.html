《游戏设计艺术》笔记

发表于2024-04-19字数统计828

游戏设计艺术

真正的发现之旅不在于寻找新风景，而是拥有新眼光。
——马塞尔·普鲁斯特

游戏设计师最终关注的是创造体验。

体验

1号透镜：情感

  • 我希望玩家能够体验到怎样的情感？为什么？
  • 当人们玩游戏时，他们产生了怎样的情感？为什么？
  • 我怎样缩小人们实际体验与我的设想之间的距离？

分析记忆

2号透镜：本质体验

  • 我想要让玩家获得怎样的体验？
  • 这种体验的本质是什么？
  • 我应该如何让我的游戏抓住这种体验的本质？

如果你对玩家获得的体验有明确的想象，并知道游戏中的哪一部分创造了这种体验，你就能更清楚地知道怎样才能把你的游戏变得更好，因为你知道游戏中哪些元素能够安全地改变，而哪些不行。

3号透镜：场景

  • 我想要制作的游戏最适合哪种场景？
  • 我的场景中是否有一些特殊的属性会影响到我的游戏？
  • 我的游戏中有哪些元素能够与场景和谐共处，哪些不行？

炉边设计的游戏系统：wii，与家庭成员一起游玩

工作台：PC（steam），单独更高，更适合单人或在线

读书角：iPad，难度更低，更简单放松

竞技场：体育运动，电子竞技

4号透镜：惊喜

  • 当玩家们玩我的游戏时，什么会让他们感到惊喜？
  • 我的游戏中的故事，规则，艺术，技术有惊喜吗？
  • 你的规则能够让玩家之间互相带来惊喜吗？
  • 你的规则能够让玩家们给自己带来惊喜吗？

5号透镜：乐趣

大多数玩乐活动都是尝试回答这样的问题：

  • 当我转动把手时会发生什么？

  • 我们能击败这支队伍吗？

  • 我能把这些黏土做成什么？

  • 我能跳神多少次？

  • 完成这个关卡后会发生什么？

6号透镜：好奇心

  • 我的游戏给玩家的大脑提出了怎样的问题？
  • 我正在做哪些努力让他们在意这些问题？
  • 我能做什么来使他们提出更多的问题？

7号透镜：内生价值

  • 在我的游戏中哪些东西对玩家很有价值？
  • 怎样让这些东西对玩家更有价值？
  • 游戏中的价值与玩家的动机之间有怎样的关系？

8号透镜：解决的问题

  • 我的游戏要求玩家解决哪些问题？
  • 是否有一些隐藏的问题作为玩法的一部分出现？
  • 在我的游戏中，怎样产生新的问题来对玩家保持吸引力？

元素

9号透镜：四种元素

四种基本元素：

* 右脑元素：美学，故事
* 左脑元素：机制，技术

10号透镜：全息设计

四种元素和玩家体验，以及联系方式。

  • 游戏中的哪一种元素产生了令人享受的体验？
  • 游戏中的哪一种元素会让玩家在体验中分心？
  • 怎样改变游戏元素来提高游戏体验？

发表于2023-09-08字数统计1.3k

Javascript

ES6

Map

JavaScript 的对象（Object），本质上是键值对的集合（Hash 结构），但是传统上只能用字符串当作键。

为了解决这个问题，ES6 提供了 Map 数据结构。它也是键值对的集合，但是“键”的范围不限于字符串，各种类型的值（包括对象）都可以当作键。

也就是说，Object 结构提供了“字符串—值”的对应，Map 结构提供了“值—值”的对应，是一种更完善的 Hash 结构实现。
